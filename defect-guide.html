기본 개념

⚡ 전기 (Lightning)
매 턴 무작위 적에게 3+밀집 뎀. 발현 시 8+밀집 뎀. 공격 구체의 핵심. 밀집(Focus) 올리면 매 턴 자동 딜이 폭증.
❄️ 냉기 (Frost)
매 턴 2+밀집 방어도. 발현 시 5+밀집 방어도. 방어 구체. 슬롯 가득 채우고 밀집 올리면 매 턴 방어도 수십.
🌑 암흑 (Dark)
매 턴 끝에 6+밀집만큼 피해 축적. 발현 시 축적된 피해를 최저 HP 적에게. 오래 키울수록 한 방이 커짐.
🔥 플라즈마 (Plasma)
매 턴 시작 시 에너지 1 획득. 발현 시 에너지 2. 밀집의 영향 없음. 에너지 경제 해결용.
Build 01

후벼 파기 스팸

★☆☆ 입문
💡 커뮤니티 최고 인기 빌드. 후벼 파기(Claw)를 반복 사용해 피해량을 영구 누적시키는 0코스트 공격 스팸 덱. 구체를 쓰지 않는 유일한 디펙트 빌드. 덱을 슬림하게 유지하고 후벼 파기를 최대한 많이 드로우해서 쓰는 게 핵심.
핵심 카드
후벼 파기 Claw 0코 공격
0코 3뎀 + 사용할 때마다 모든 후벼 파기 피해 +2 영구 증가. 강화 시 +3. 이 빌드의 전부. 3장 이상 모이면 올인
하나를 위한 모두 All for One 레어
2코 10뎀 + 버린 카드 더미의 모든 0코스트 카드를 손으로. 후벼 파기 전부 회수. 이 빌드의 윈컨
긁어내기 Scrape 고급
1코 7뎀 + 카드 4장 드로우. 0코 아닌 카드는 버림. 후벼 파기 찾기 최적화
추진 타격 Momentum Strike
1코 10뎀 + 비용이 0으로 감소. 첫 사용 후 0코 → 하나를 위한 모두로 회수 가능
FTL FTL 고급
0코 5뎀 + 카드 3장 미만 사용했으면 1장 드로우. 턴 초반에 쓰면 공짜 드로우
서포트
야성 Feral 파워
2코 파워. 매 턴 처음 사용한 0코 공격이 손으로 복귀. 메아리의 형상 저가 버전. 후벼 파기 무한 루프
홀로그램 Hologram
1코 방어도 3 + 버린 카드에서 1장 회수. 강화 시 소멸 제거. 하나를 위한 모두 재활용
걷어내기 Skim 고급
1코 카드 3장 드로우. 강화 시 4장. 후벼 파기 찾기용 순수 드로우
이상적인 턴 (후벼 파기 3장+ 보유 시)
  1. 후벼 파기 × 3 (0코) → 각 +2뎀 누적 = 총 피해 폭증
  2. 하나를 위한 모두 (2코) → 버린 0코 카드 전부 회수
  3. 후벼 파기 × 3 다시 → 총 6회 사용, 피해 +12 누적
  4. 야성 있으면 다음 턴에도 자동 복귀 → 무한 스노우볼
주의: 이 빌드는 구체를 쓰지 않아. 구체 시너지 유물(미라의 손 등)은 아무 효과 없으니 유물 선택 신중하게. 카드 수를 15~20장 이하로 유지해서 후벼 파기 드로우 확률을 높이는 게 생명.
추천 유물
쌍절곤 — 공격 10장 사용 시 에너지 1. 후벼 파기 스팸으로 자연 달성
사슬낫 — 공격 3장 사용마다 적에게 6뎀. 0코 스팸과 완벽한 시너지
수리검 — 공격 3장 사용 시 힘 +1. 후벼 파기 포함 모든 공격 강화

냉기 구체 방벽

★☆☆ 입문
냉기 구체를 슬롯에 가득 채우고 밀집(Focus)을 올려서 매 턴 자동으로 방어도를 수십씩 쌓는 철벽 빌드. 가장 안정적인 디펙트 빌드로, 긴 전투에서 절대 죽지 않는 탱커가 됨.
핵심 카드
빙하 Glacier 고급
2코 방어도 6 + 냉기 2번 영창. 방어도+냉기 동시. 이 빌드의 주력
냉정함 Coolheaded
1코 냉기 1번 영창 + 1장 드로우. 강화 시 2장 드로우. 냉기+순환 동시
꽃샘추위 Cold Snap
1코 6뎀 + 냉기 1번 영창. 딜+냉기 동시. 초반 주력
오한 Chill 고급
0코 적 하나당 냉기 1번 영창. 소멸. 강화 시 소멸 제거. 다적 전투에서 냉기 대량 생성
조각모음 Defragment 파워
1코 밀집 +1. 강화 시 +2. 모든 구체 패시브와 발현 강화. 구체 빌드 필수
축전기 Capacitor 파워
1코 구체 슬롯 +2. 강화 시 +3. 냉기 더 많이 유지 = 방어도 더 많이
밀집 부스트
인지 편향 Biased Cognition 고대의 존재
1코 밀집 +4. 매 턴 밀집 -1. 밀집 4를 한 번에. 유물로 감소 상쇄 가능
냉각재 Coolant 레어 파워
1코 파워. 매 턴 영창된 구체 종류당 방어도 2. 냉기+전기면 매 턴 4 추가 방어도
반복 Loop 파워
1코 파워. 매 턴 가장 오른쪽 구체 지속 능력 추가 발동. 강화 시 2회. 냉기면 방어도 2배
왜 강하냐면: 냉기 3개 + 밀집 5면 매 턴 패시브 방어도 = 3 × (2+5) = 21. 반복 강화까지 있으면 가장 오른쪽 냉기가 3번 발동 = 추가 14. 총 매 턴 35+ 자동 방어도. 카드 안 써도 돼.
추천 유물
데이터 디스크 — 매 전투 시작 시 밀집 +1. 디펙트 전용. 필수급
캘리퍼스 — 방어도 15 이상 유지. 냉기 방어도 다음 턴으로 이월
핵 — 시작 시 전기 3번 영창. 부서진 핵 대체. 초반 안정성 폭증
Build 03

전기 폭풍

★★☆ 중급
전기 구체를 대량 영창하고 빠르게 발현시켜 매 턴 적 전체에 번개를 퍼붓는 공격적인 빌드. 구체 슬롯을 적게 유지해서 전기가 바로 발현되게 하는 게 핵심. 벼락+동전기 콤보가 완성되면 전기 하나당 추가 뎀까지 들어감.
핵심 카드
번개 구체 Ball Lightning
1코 7뎀 + 전기 1번 영창. 딜+전기 동시. 초반 주력
테슬라 코일 Tesla Coil 고급
0코 3뎀 + 대상 적에게 모든 전기 발동. 전기 5개면 한 번에 5번 발동
동전기 Voltaic 레어
2코 이번 전투 동안 영창한 전기 수만큼 전기 영창. 소멸. 후반부에 쓰면 전기 10개+
벼락 Thunder 파워
1코 파워. 전기 발현할 때마다 적에게 추가 6뎀. 강화 시 8뎀. 전기 발현 보너스
폭풍 Storm 파워
1코 파워. 파워 카드 사용할 때마다 전기 1번 영창. 강화 시 2번. 파워 스팸=전기 스팸
산산조각 Shatter 레어
1코 전체 11뎀 + 모든 구체 발현. 강화 시 15뎀. 전기 가득 찼을 때 쓰면 폭딜
폭딜 턴 (벼락+폭풍 설치 후)
  1. 폭풍 효과로 파워 쓸 때마다 전기 자동 영창
  2. 동전기 → 전투 동안 영창한 전기 수만큼 전기 대량 영창
  3. 산산조각 → 모든 구체 발현 + 벼락 효과로 발현당 추가 6~8뎀
  4. 전기 10개 발현 = 80+뎀 AoE + 산산조각 11뎀 = 90+뎀
추천 유물
데이터 디스크 — 밀집 +1 = 전기 패시브 4뎀, 발현 9뎀으로 증가
금강저 — 힘 +1. 전기와 별개로 공격 카드 피해 증가
핵 — 시작 시 전기 3번. 1턴부터 전기 패시브 자동 발동

암흑 구체 원펀치

★★☆ 중급
암흑 구체를 영창하고 냉기 방벽 뒤에서 천천히 피해를 축적시킨 뒤, 다중 시전이나 사중 시전으로 한 번에 터뜨리는 원샷 빌드. 보스전에서 특히 강력. 냉기로 버티면서 암흑이 자랄 시간을 벌어주는 하이브리드 전략.
핵심 카드
어둠 Darkness 고급
1코 암흑 1번 영창 + 모든 암흑 패시브 발동. 강화 시 2회 발동. 암흑 성장 가속
그림자 방패 Shadow Shield 고급
2코 방어도 11 + 암흑 1번 영창. 방어+암흑 동시. 안정적인 셋업
그림자 소모 Consuming Shadow 레어 파워
2코 파워. 암흑 2번 영창 + 매 턴 끝 가장 왼쪽 구체 발현. 강화 시 3번. 자동 발현 엔진
다중 시전 Multi-Cast 레어
X코 가장 오른쪽 구체를 X번 발현. 암흑 축적 100+ → X=3이면 300+뎀
사중 시전 Quadcast 고대의 존재
1코 가장 오른쪽 구체를 4번 발현. 강화 시 0코. 암흑 200 축적 → 800뎀. 보스 원킬
방어+스톨 (암흑 성장 시간 벌기)
빙하 Glacier
냉기 2번 영창. 암흑 앞에 냉기 벽 세워서 발현 방지+방어도 확보
냉정함 Coolheaded
냉기 영창+드로우. 구체 슬롯을 냉기로 채워서 암흑을 보호
축전기 Capacitor
슬롯 +2. 암흑이 뒤에서 자랄 공간 확보 + 냉기 추가 유지
왜 강하냐면: 암흑은 매 턴 6+밀집만큼 축적. 밀집 5면 매 턴 11씩 증가. 5턴이면 55 축적. 여기에 어둠 카드로 패시브 추가 발동하면 10턴에 200+ 축적. 다중 시전 X=3이면 600+뎀 원샷. 보스도 한 방.
추천 유물
데이터 디스크 — 밀집 +1 = 암흑 매 턴 축적량 +1. 10턴이면 +10
캘리퍼스 — 냉기 방어도 이월. 암흑 키우는 동안 버티기 쉬워짐
공생 바이러스 — 전투 시작 시 암흑 자동 영창. 1턴부터 축적 시작

메아리의 형상 엔진

★★★ 고급
💡 고점 최강 빌드. 메아리의 형상으로 매 턴 첫 카드를 2번 사용. 조각모음이 2번 = 밀집+2, 빙하가 2번 = 냉기 4개+방어도 12. 모든 카드의 가치가 2배가 되니 뭘 해도 강함. 설치 비용 3코+휘발성이 문제지만, 완성되면 무적.
핵심 카드
메아리의 형상 Echo Form 레어 파워
3코 파워. 휘발성. 매 턴 처음 사용하는 카드가 1번 추가 사용. 강화 시 휘발성 제거. 이 빌드의 전부
조각모음 Defragment
메아리의 형상 → 조각모음 2번 사용 = 밀집 +2. 강화 시 +4. 매 턴 구체 폭강
빙하 Glacier
메아리 → 빙하 2번 = 냉기 4개 + 방어도 12. 한 장으로 방벽 완성
버퍼 Buffer 레어 파워
2코 파워. 다음 체력 손실 무효화. 강화 시 2회. 메아리로 2번 = 4회 면역
에너자이저 Double Energy 레어
1코 에너지 2배. 소멸. 메아리의 형상 설치 턴에 에너지 확보용
셋업 보조
부팅 과정 Boot Sequence 고급
0코 선천성 방어도 10. 소멸. 메아리의 형상 설치 턴 방어도 확보
임계 초과 Supercritical 레어
0코 에너지 4. 소멸. 메아리의 형상(3코) 설치 후 남은 에너지로 추가 행동
신호 증폭 Signal Boost 레어
1코 다음 파워가 1번 추가 사용. 소멸. 메아리의 형상 전에 쓰면 메아리 2장 효과
완성 후 매 턴 루틴
  1. 첫 카드 조각모음 → 2번 발동 = 밀집 +2 (매 턴)
  2. 또는 빙하 → 냉기 4개 = 매 턴 방어도 28+ (밀집 5 기준)
  3. 또는 버퍼 → 면역 4회. 사실상 무적
  4. 메아리 2개 설치하면 첫 2장이 2번 → 완전한 신

창의적인 인공지능 파워 스팸

★★☆ 중급
💡 자체 분석 추천. 창의적인 인공지능으로 매 턴 무작위 파워를 받아 끝없이 버프를 쌓는 빌드. 폭풍과 조합하면 파워 쓸 때마다 전기 영창. 미라의 손 유물이 있으면 파워 사용 시 무작위 카드 비용 0 → 무한 엔진.
핵심 카드
창의적인 인공지능 Creative AI 레어 파워
3코 파워. 매 턴 무작위 파워 카드 1장 손으로. 강화 시 2코. 무한 버프 엔진
폭풍 Storm 파워
파워 쓸 때마다 전기 영창. 창의적인 인공지능이 매 턴 파워 주니까 = 매 턴 전기 자동 영창
서브루틴 Subroutine 파워
1코 파워. 파워 사용할 때마다 에너지 1. 강화 시 0코. 파워 스팸 = 에너지 무한
버퍼 Buffer 레어 파워
생존기. 창의적인 인공지능이 버퍼를 줄 수도 있어서 무한 면역 가능
기계학습 Machine Learning 레어 파워
1코 파워. 매 턴 카드 1장 추가 드로우. 파워 스팸으로 받은 카드를 더 빨리 사용
백색 소음 White Noise 고급
1코 무작위 파워 1장 + 이번 턴 무료 사용. 소멸. 강화 시 0코. 파워 서치+설치
왜 강하냐면: 창의적인 인공지능이 10턴 돌면 파워 10장을 공짜로 받은 셈. 조각모음, 반복, 축전기, 벼락, 버퍼 같은 강력한 파워가 무작위로 쌓이면 밀집 10+, 구체 슬롯 8+, 면역 3+ 같은 미친 상태가 됨. 미라의 손 유물(파워 사용 시 무작위 카드 비용 0)이 있으면 파워 → 공짜 카드 → 공짜 파워 → 무한.

상태이상 연료 엔진

★★☆ 중급
💡 자체 분석 추천. 화상, 어지러움, 공허 같은 상태이상 카드를 의도적으로 생성하고, 이를 자원으로 변환하는 역발상 빌드. 굴뚝(상태이상 생성 시 전체 5뎀), 고철을 보물로(상태이상 시 구체 영창), 순회(상태이상 드로우 시 추가 드로우)로 디메리트를 메리트로.
상태이상 생성
오버클럭 Overclock 고급
0코 카드 2장 드로우 + 화상 1장 생성. 강화 시 3장 드로우. 공짜 드로우+상태이상 트리거
가속 이탈 Boost Away
0코 방어도 6 + 어지러움 1장 생성. 0코 방어+상태이상 트리거
틀어막기 Gunk Up
1코 4뎀 ×3 + 점액투성이 1장 생성. 멀티히트+상태이상
강행 돌파 Fight Through 고급
1코 방어도 13 + 부상 2장 생성. 높은 방어도+상태이상 2개 트리거
로켓 펀치 Rocket Punch 고급
2코 13뎀 + 드로우 1 + 상태이상 생성 시 이번 턴 비용 0. 상태이상 빌드면 공짜
상태이상 활용 (페이오프)
굴뚝 Smokestack 파워
1코 파워. 상태이상 생성할 때마다 전체 5뎀. 강화 시 7뎀. 이 빌드의 딜 엔진
고철을 보물로 Trash to Treasure 레어 파워
1코 파워. 상태이상 생성마다 무작위 구체 1번 영창. 상태이상=구체=추가 가치
순회 Iteration 파워
1코 파워. 매 턴 처음 상태이상 드로우 시 카드 2장 추가 드로우. 강화 시 3장
대공포 Flak Cannon 레어
2코 모든 상태이상 소멸 + 1장당 무작위 적 8뎀. 상태이상 5장이면 40뎀 버스트
압축 Compact 고급
1코 방어도 6 + 손의 상태이상을 연료로 변환. 상태이상 정리+에너지 확보
왜 강하냐면: 오버클럭+가속 이탈+강행 돌파를 한 턴에 쓰면 상태이상 4장 생성. 굴뚝 있으면 = 전체 20뎀 자동. 고철을 보물로까지 있으면 = 구체 4개 영창. 대공포로 상태이상 정리하면 추가 32뎀. 디메리트가 딜+구체+드로우로 변환되는 연금술.

유리 구체 폭발

★★★ 고급
💡 자체 분석 추천 — 신규 메카닉. 슬더스 2의 새로운 유리(Glass) 구체를 활용하는 빌드. 유리 구체는 발현 시 피해를 주고, 스피너(매 턴 유리 영창)와 굴절(유리 2번 영창+공격)을 축으로 유리를 대량 생산해서 폭발적인 딜을 터뜨린다.
핵심 카드
유리 공예 Glasswork 고급

굴절 Refract 레어
3코 9뎀 ×2 + 유리 2번 영창. 강화 시 12뎀. 대량 유리 생산+딜 동시
스피너 Spinner 레어 파워
1코 파워. 매 턴 유리 1번 영창. 강화 시 카드 사용 시에도 영창. 유리 자동 생산
산산조각 Shatter 레어
1코 전체 11뎀 + 모든 구체 발현. 유리 가득 + 산산조각 = 유리 전부 폭발
동기화 Synchronize 고급
1코 영창된 구체 종류당 밀집 2. 소멸. 유리+전기+냉기 = 밀집 6 한 번에

무지개 Rainbow 레어
2코 전기+냉기+암흑 각 1번 영창. 소멸. 강화 시 소멸 제거. 다양한 구체로 동기화 극대화
혼돈 Chaos
1코 무작위 구체 영창. 강화 시 2번. 구체 종류 다양성 확보

초반 운영 팁

1 밀집(Focus)이 핵심. 밀집 1 올리면 모든 구체의 패시브+발현이 강해짐. 구체 빌드면 조각모음을 최우선으로 챙겨. 밀집 없는 구체는 허수아비야.
2 Act 1은 빌드 고르지 마. 후벼 파기가 2장 뜨면 후벼 파기 덱, 구체 카드가 몰리면 구체 빌드. 뜨는 카드에 유연하게 대응하는 게 맞아.
3 구체 슬롯 관리는 의도적으로. 공격적 빌드(전기)는 슬롯 적게 → 빠른 발현. 방어적 빌드(냉기)는 슬롯 많이 → 패시브 극대화. 목적에 맞게 조절해.
4 부팅 과정과 오한은 초반 생명줄. 디펙트는 1턴이 항상 약해. 부팅 과정(0코 방어도 10)이나 오한(0코 냉기 영창)으로 초반 안정성 확보. 둘 다 잡지 말고 하나만.
5 후벼 파기와 구체를 섞지 마. 후벼 파기 덱은 구체를 안 쓰고, 구체 덱은 후벼 파기가 필요 없어. 한쪽에 올인해야 승률이 올라.
6 인지 편향은 양날의 검. 밀집 +4는 사기지만 매 턴 -1이라 장기전에서 밀집이 음수가 됨. 짧은 전투용이거나, 유물로 디버프 제거가 가능할 때만 써.
7 덱을 슬림하게 유지해. 디펙트는 핵심 카드 시너지가 중요해서 덱이 두꺼우면 핵심 조합이 안 맞아. 카드 제거를 적극 활용하고, 불필요한 카드는 스킵해.
